Features

Everything you need to orchestrate

Purpose-built for developers who manage complex AI workflows across multiple repositories and branches.

Multi-Agent Orchestration

Run multiple AI agents in parallel across workspaces. Coordinate tasks, share context, and merge results seamlessly.

Automated Code Review

Full diff viewing with file tree navigation. AI-powered review suggestions on every commit, branch, and pull request.

Git Integration

Deep git operations: diff parsing, merge strategies, conflict detection, and branch management built into every workflow.

Integrated Terminal

Full PTY terminal with session caching. Execute commands, inspect output, and manage processes without leaving the app.

Security First

Trust dialogs for command execution, secure keychain credential storage, and sandboxed agent environments by default.

Blazing Performance

Rust backend with Tokio async runtime. Tauri's lightweight footprint means minimal memory usage and instant startup.

agent.rs
pub async fn orchestrate( agents: Vec<AgentConfig>, workspace: &Workspace, ) -> Result<Vec<AgentResult>> { // Launch agents in parallel let handles = agents .iter() .map(|a| spawn_agent(a, workspace)) .collect::<Vec<_>>(); // Await all results with timeout let results = join_all(handles) .await .into_iter() .collect::<Result<_>>()?; Ok(results) }
